## _Testing_
---
This project uses JUNIT 5 for testing. To run the tests follow the steps below:
```bash
# make sure you are in the cookbook-project directory
cd cookbook-project #(if not)

# navigate to the ui or core directory
cd ui 
# or
cd core

# run tests
mvn test

```
